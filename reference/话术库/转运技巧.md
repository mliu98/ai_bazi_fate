# 🔮 转运技巧话术库 · 完整版 v1.0

> **设计规则**：每次取 5 条，按「缺失五行（1-2条）+ 时间仪式（1条）+ 颜色/方位（1条）+ TIER专属（1条）」组合输出。
> **字段说明**：每条含 `id`（唯一索引）、`wx_tag`（五行标签）、`tier_tag`（缘分值标签，`ALL`=通用）、`category`（类别）、`title`（标题，20字以内）、`body`（正文，60-100字）
> **质检要求**：语气轻松不玄乎，必须有实操建议，不出现「一定会成功」「保证在一起」等绝对承诺。

## 五、拼装规则（开发参考）

### 5.1 转运技巧选取逻辑

```
每次取 5 条（优化自原4条规则）：
1. 缺失五行补益 × 1-2 条
   → 根据合盘五行计算，找出缺失的五行
   → 从对应补益池（ZY_W / ZY_MU / ZY_JIN / ZY_HUO / ZY_TU）取话术
   → 若缺两种五行，各取1条

2. 时间仪式 × 1 条
   → 按 TIER_TAG 匹配（TIER_S取RI01-RI05，TIER_B取RI06-RI10，TIER_C/D取RI11-RI15）
   → 同一 tier 随机取1条

3. 颜色约会 × 1 条
   → 按 WX_TAG 匹配（CL01-CL07）
   → 精准匹配五行关系

4. 方位或数字 × 1 条
   → TIER_S/A：优先取 SZ01 或 SZ02（数字专属）
   → TIER_B/C：优先取 FW01（桃花位）
   → TIER_D：取 FW05（出行时机建议）
```

### 5.2 数据库字段说明

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | VARCHAR(12) | 主键，如 ZY_W01 |
| wx_tag | VARCHAR(100) | 五行标签，逗号分隔，ALL=通用 |
| tier_tag | VARCHAR(50) | 缘分值区间，逗号分隔，ALL=通用 |
| category | VARCHAR(30) | 类别（补益·水/仪式·时间/颜色·约会/方位·桃花位/数字·专属） |
| title | VARCHAR(50) | 话术标题 |
| body | TEXT | 话术正文，含 {A} {B} 占位符 |
| created_at | TIMESTAMP | 创建时间 |

### 5.3 话术变量替换规则

- `{A}` → 用户（提测者）姓名
- `{B}` → 对方姓名
- `{color_A}` → 按{A}五行属性：木=绿色、火=红色、土=黄/棕色、金=白/金色、水=蓝/黑色
- `{color_B}` → 按{B}五行属性，同上
